# Product Overview

MindMelee is a real-time AI debate coaching application that helps users practice and improve their debating skills through voice interaction with an AI opponent powered by Google's Gemini Live API.

## Core Features

- Real-time voice debate with AI opponent using bidirectional audio streaming
- Two debate modes: Coach (supportive, educational) and Fierce (aggressive, challenging)
- Comprehensive performance analytics with 10+ metrics
- Session history tracking with local persistence
- UI-based settings for API key and model configuration
- Support for multiple Gemini models (2.0, 2.5, 3.0)

## User Flow

1. User configures API key and models via Settings UI
2. User selects debate topic, style (Coach/Fierce), and duration on Dashboard
3. Live debate session with real-time audio streaming and transcription
4. Post-debate analysis with detailed performance metrics and feedback
5. Results saved to local history for progress tracking

## Target Audience

Individuals looking to improve their debate and argumentation skills through AI-powered practice sessions with immediate feedback.
